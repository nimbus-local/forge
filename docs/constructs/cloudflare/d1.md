# D1Database

Creates a Cloudflare D1 SQLite database.

```go
import cf "github.com/nimbus-local/forge/constructs/cloudflare"

db := cf.NewD1Database(ctx, "Users", nil)
```

---

## D1DatabaseArgs

`D1DatabaseArgs` has no fields — the database name and account ID are derived from the construct name and `AppConfig.Cloudflare`.

---

## Methods

```go
func (d *D1Database) ID() pulumi.IDOutput        // database UUID
func (d *D1Database) Name() pulumi.StringOutput  // physical database name (qualified)
```

---

## Linkable

| Env var | Value |
|---|---|
| `SST_D1_<NAME>_ID` | D1 database UUID |
| `SST_D1_<NAME>_NAME` | Physical database name |

---

## Binding to a Worker

Pass the `*D1Database` in `WorkerArgs.D1Bindings`. The binding variable name is the SCREAMING_SNAKE_CASE of the construct name:

```go
db := cf.NewD1Database(ctx, "Users", nil)

worker := cf.NewWorker(ctx, "Api", &cf.WorkerArgs{
    Handler:    "worker/index.ts",
    D1Bindings: []*cf.D1Database{db},
})
```

Inside the worker:

```ts
const result = await env.USERS
    .prepare("SELECT * FROM users WHERE id = ?")
    .bind(userId)
    .first();
```

---

## Running migrations

D1 migrations are managed separately via the Cloudflare dashboard or `wrangler d1 migrations apply`. forge creates the database but does not run migrations.

---

## Linking to an AWS Lambda

```go
lambda := constructs.NewFunction(ctx, "Sync", &constructs.FunctionArgs{
    Handler: "bootstrap",
    Link:    []forge.Linkable{db},
})
```

Inside the Lambda (use Cloudflare REST API with the database UUID):

```go
dbID   := os.Getenv("SST_D1_USERS_ID")
dbName := os.Getenv("SST_D1_USERS_NAME")
```
