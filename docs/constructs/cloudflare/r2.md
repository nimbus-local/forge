# R2Bucket

Creates a Cloudflare R2 object-storage bucket.

```go
import cf "github.com/nimbus-local/forge/constructs/cloudflare"

bucket := cf.NewR2Bucket(ctx, "Assets", nil)
```

---

## R2BucketArgs

| Field | Type | Default | Description |
|---|---|---|---|
| `Location` | `string` | `""` | Storage location hint: `"enam"` (Eastern NA), `"weur"` (Western EU), `"apac"`. Omit to let Cloudflare choose. |

---

## Methods

```go
func (r *R2Bucket) Name() pulumi.StringOutput  // physical bucket name (qualified)
```

---

## Linkable

| Env var | Value |
|---|---|
| `SST_R2_<NAME>_NAME` | Physical R2 bucket name |

---

## Binding to a Worker

Pass the `*R2Bucket` in `WorkerArgs.R2Bindings`. The binding variable name is the SCREAMING_SNAKE_CASE of the construct name:

```go
bucket := cf.NewR2Bucket(ctx, "Assets", &cf.R2BucketArgs{
    Location: "weur",
})

worker := cf.NewWorker(ctx, "Uploader", &cf.WorkerArgs{
    Handler:    "worker/index.ts",
    R2Bindings: []*cf.R2Bucket{bucket},
})
```

Inside the worker:

```ts
// Upload
await env.ASSETS.put("images/logo.png", request.body, {
    httpMetadata: { contentType: "image/png" },
});

// Download
const obj = await env.ASSETS.get("images/logo.png");
if (!obj) return new Response("Not Found", { status: 404 });
return new Response(obj.body);
```

## Example — geo-located media bucket

```go
euBucket := cf.NewR2Bucket(ctx, "EuMedia", &cf.R2BucketArgs{Location: "weur"})
usBucket := cf.NewR2Bucket(ctx, "UsMedia", &cf.R2BucketArgs{Location: "enam"})

worker := cf.NewWorker(ctx, "MediaRouter", &cf.WorkerArgs{
    Handler: "worker/router.ts",
    R2Bindings: []*cf.R2Bucket{euBucket, usBucket},
})
```
